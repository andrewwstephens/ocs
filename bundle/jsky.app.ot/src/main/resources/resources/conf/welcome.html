Welcome to Observing Tool

This is the Gemini Observing Tool (OT), which is used to develop and plan Phase 2 Science Programs for the Gemini telescopes.

Click below on the appropriate button to create a New Program, to Open an existing local or remote program, to Manage your keychain, to search the database with the OT Browser, or to Import a program from XML.

The Observing Tool is the work of the Gemini Observatory staff including Bryan Miller, Andrew Stephens, and the software engineers Arturo Nunez, Sebastian Raaphorst, Carlos Quiroz, Rob Norris, Shane Walker, and many former employees.

The image at left shows Jupiter at 4.7 microns taken with NIRI on Gemini North. Many short images were taken to "freeze" the blurring due to the Earth's atmosphere and the sharpest 10% were combined to produce this image. It is the highest resolution infrared image of Jupiter taken from the ground. The bright regions are the glow from warmer regions in the interior that are not blocked by higher clouds.

Contact Bryan Miller (bmiller@gemini.edu) for more information.

The Observing Tool uses Java code from a number of other projects and we would like to thank them for their work.

  • Jeremy Jones and his team at the NASA Goddard Space Flight Center provided the WCS, TimeLine and utility classes from SEA that were modified and used in JSky and OT.
The following external packages are used by parts of JSky and OT (the jar files are included in the source release):
  • JFreeChart - an open-source class library for producing charts.
  • Diva - Dynamic Interactive Visualization toolkit (used for interactive image graphics).
  • JEL - Java Expression Language, a Java expression compiler (used to evaluate catalog plot symbol expressions based on column values).
  • FITS - Tom McGlynn's FITS I/O library.
  • H-Compress - Patrick Dowler's FITS H-Compress library.
  • Log4J - Message logging facility.
  • dom4j - Java support for reading/writing XML files.
  • JUNIT - Java support for unit testing.
  • MyDoggy - Java docking framework to be used in cross-platform Swing applications.
